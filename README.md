# 🛒 AkıllıMarket - Akıllı Alışveriş Asistanı

Modern, kullanıcı dostu ve AI destekli Progressive Web App (PWA) market uygulaması.

![AkıllıMarket Banner](https://img.shields.io/badge/PWA-Ready-blue) ![Version](https://img.shields.io/badge/version-2.3.0-green) ![License](https://img.shields.io/badge/license-MIT-blue)

## ✨ Özellikler

### 🤖 AI Destekli Yemek Planlaması
- **Akıllı tarif önerileri**: 25+ farklı yemek tarifi
- **Otomatik malzeme listesi**: Seçtiğiniz tariflerin malzemelerini otomatik ekler
- **Kategori bazlı arama**: Türk mutfağı, dünya mutfağı, tatlılar
- **Aşçı asistanı**: AI ile yemek önerileri

### 📝 Akıllı Alışveriş Listesi
- **Otomatik kategorizasyon**: Ürünler otomatik olarak kategorilere ayrılır
- **Emoji desteği**: Her ürün için otomatik emoji
- **Çevrimdışı çalışma**: İnternet olmadan da kullanabilirsiniz
- **Akıllı bildirimler**: Alışveriş hatırlatıcıları

### 📱 PWA Özellikleri
- **Ana ekrana ekleme**: Native app gibi çalışır
- **Çevrimdışı mod**: Service Worker ile offline çalışma
- **Responsive tasarım**: Tüm cihazlarda mükemmel görünüm
- **Dark mode**: Göz dostu koyu tema
- **Accessibility**: Görme zorluğu olan kullanıcılar için özellikler

## 📱 Kurulum

### Web Tarayıcısında
1. [Demo linkini](https://firatfurkan.github.io/market-app-pwa/) açın
2. Tarayıcınızın "Ana ekrana ekle" önerisini kabul edin
3. Uygulama artık telefonunuzda native app gibi çalışacak

## 🎯 Kullanım

### 1. Giriş Yapın
- Kullanıcı adınızı girin
- Güven puanınız otomatik olarak oluşturulur

### 2. AI Aşçı Asistanını Kullanın
- "makarna" yazın, malzemeler listelenir
- "tavuklu pilav" yazın, tarif önerileri gelir
- "pizza" yazın, gerekli malzemeler hazırlanır

### 3. Alışveriş Listesi Oluşturun
- Manuel ürün ekleme (miktar + birim)
- AI önerilerini toplu ekleme
- Kategorilere göre düzenli liste
- Çöp kutusu ile ürün silme

## 🛠️ Teknolojiler

- **Frontend**: Vanilla JavaScript, HTML5, CSS3
- **PWA**: Service Worker, Web App Manifest
- **Depolama**: LocalStorage
- **Tasarım**: CSS Grid, Flexbox, CSS Variables
- **Hosting**: GitHub Pages
- **Font**: Google Fonts (Poppins)
